# SWEST20 プログラム

<a name="panel"></a>
## 基調パネルセッション

* **テーマ:** `future = SWEST(&you) + dream;`  
**“組込み”**の関数定義と機能を議論して、**“あなた”**が抱く**“夢”**とともに語り合うと、どのような**“未来”**を拓けるでしょう？  
＝＋回目の開催となるSWESTを記念して、これまで基調講演いただいた方々をパネラとしてお招きし、参加者の皆さまと組込みのこれまでとこれからについて、徹底的に議論します！
